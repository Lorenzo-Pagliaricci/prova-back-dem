# Backstage Overview

Questa piattaforma Backstage gestisce il provisioning end-to-end dei nuovi
servizi.

## Cosa fa in generale

- crea il repository su GitLab;
- genera i file base del progetto;
- crea e configura il job Jenkins;
- avvia la prima build della pipeline;
- pubblica gli artifact su Nexus.

## Risultato finale

Dopo la creazione, il componente e visibile nel Catalog con le tab principali:

- Overview
- GitLab
- CI/CD
- Artifact
- Docs
